# Usages

Import InputsModule into your custom module

```typescript
import { SwitchModule } from '@atoms/switch/switch.module';

@NgModule({
    imports: [SwitchModule]
})

export class YourModule {}
```

## Examples

  ```html
    <app-switch [disabled]="true"></app-switch>
```
 ```html
    <app-switch [checked]="true"></app-switch>
  ```
