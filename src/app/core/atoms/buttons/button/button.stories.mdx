# Button
<br/>
<br/>

### Props
| Property | Type  | Values | Default  |
| ------ | --------- |------ | --------- |
| appearance    | string   | <code>primary</code><code>secondary</code><code>outline</code><code>ghost</code><code>default</code> | <code>default</code> |
| behavior      | string   | <code>button</code><code>submit</code><code>reset</code> | <code>button</code> |
| disabled      | boolean  | true/false       | false     |
| inlineStyle   | object/string |   ""     | ""  |
| innerClass    | string   |     ""    |  ""  |

### Events
| Event | Arguments  | Description |
| ------ | --------- |------ |
| click    |     | Emitted when clicked on button |

------

# Usages

Import ButtonsModule into your custom module

```typescript
import { ButtonsModule } from '@atoms/buttons/buttons.module';

@NgModule({
    imports: [ButtonsModule]
})

export class YourModule {}
```

## Examples

  ```html
<app-button type="primary"
            behavior="submit"
            [disabled]="true"
            (click)="event()"></app-button>
```
 ```html
<app-button innerClass="text-white"
            [disabled]="false"
            inlineStyle="{backgroundColor: black}"></app-button>
  ```
