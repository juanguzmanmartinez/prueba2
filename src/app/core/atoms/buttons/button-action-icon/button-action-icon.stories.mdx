# Button Action Icon
<br/>
<br/>

### Props
| Property | Type  | Values | Default  |
| ------ | --------- |------ | --------- |
| iconName*    | string   | "" | "" |
| disabled      | boolean  | true/false       | false     |
| inlineStyle   | object/string |   ""     | ""  |
| innerClass    | string   |     ""    |  ""  |
*Required

### Events
| Event | Arguments  | Description |
| ------ | --------- |------ |
| click    |     | Emitted when clicked on button |

------

# Usages

Import ButtonsModule into your custom module

```typescript
import { ButtonsModule } from '@atoms/buttons/buttons.module';

@NgModule({
    imports: [ButtonsModule]
})

export class YourModule {}
```


## Examples

  ```html
     <app-button-action-icon iconName="edit"
                             (click)="event()"
                             [disabled]="true"></app-button-action-icon>
```
 ```html
     <app-button-action-icon iconName="launch"
                             [inlineStyle]="{width: '32px', height: '32px'}"
                             innerClass="cursor-pointer"></app-button-action-icon>
  ```
