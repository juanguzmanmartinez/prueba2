# Button Icon
<br/>
<br/>

### Props
| Property | Type  | Values | Default  |
| ------ | --------- |------ | --------- |
| fontName    | string   | "" | "" |
| fontColor    | string   | "" | "" |
| fontSize      | string   | <code>16px</code><code>24px</code><code>32px</code><code>48px</code> | <code>16px</code> |
| svgName    | string   | "" | "" |
| svgWidth      | string   | <code>16px</code><code>24px</code><code>32px</code><code>48px</code> | <code>16px</code> |
| svgHeight      | string   | <code>16px</code><code>24px</code><code>32px</code><code>48px</code> | null |
| innerClass    | string   |     ""    |  ""  |

*Use only <code>font</code> or <code>svg</code> property.


------

# Usages
Import ButtonsModule into your custom module

```typescript
import { IconsModule } from '@atoms/icons/icons.module';

@NgModule({
    imports: [IconsModule]
})

export class YourModule {}
```

## Examples

  ```html
<app-icon svgWidth="16" svgName="calendar"></app-icon>
<app-icon svgWidth="16" svgName="moto"></app-icon>
<app-icon svgWidth="56" svgName="datepicker"></app-icon>
<app-icon svgWidth="14" svgName="search"></app-icon>
<app-icon fontSize="16" fontName="edit"></app-icon>
<app-icon fontSize="24" fontName="launch"></app-icon>
<app-icon fontSize="32" fontName="assignment"></app-icon>
  ```

# Register new icons
<br/>

#### 1.Add icon inside these directories

  ````
  src/assets/icons/
  src/assets/illustrations/
  ````

#### 2.Update constant file

<code>src/app/core/atoms/icons/constants/icons-svg.ts</code>

   ````typescript

const ICONS_SVG = [
{...},
{name: 'ICON_NAME', path: `${path}ICON_NAME.svg`},
{name: 'ILLUSTRATION_NAME', path: `${pathIllustrations}ILLUSTRATION_NAME.svg`},
];
   ````

#### 3.Update README.md/icon.story.mdx

Update file with new icons
