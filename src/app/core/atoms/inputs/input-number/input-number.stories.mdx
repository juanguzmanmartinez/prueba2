# Input
<br/>
<br/>

### Props
| Property | Type  | Values | Default  |
| ------ | --------- |------ | --------- |
| id    | string   |  | "Input" |
| name    | string   |  | "Input" |
| behavior      | string   | <code>number</code><code>text</code><code>search</code><code>password</code> | <code>text</code> |
| placeholder    | string   |  | "placeholder" |
| innerClass    | string   |     ""    |  ""  |
| labelClass    | string   |     ""    |  ""  |
| error      | boolean  | true/false       | false     |

### Events
| Event | Arguments  | Description |
| ------ | --------- |------ |
| blur    |     | Emitted when input lost focus |
| focus    |     | Emitted when input received focus |
| change    |     | Emitted when input's value changes |

# Usages

Import InputsModule into your custom module

```typescript
import { InputsModule } from './inputs.module';

@NgModule({
    imports: [InputsModule]
})

export class YourModule {}
```

## Examples

  ```html
<app-input-number placeholder="Write something..."></app-input-number>
```
 ```html
<app-input-number value="example" [error]="true"></app-input-number>
  ```
