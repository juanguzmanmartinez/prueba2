### Props
| Property | Type  | Values | Default  |
| ------ | --------- |------ | --------- |
| router    | any[] \ string |  |  |
| routerActive      | string[] \ string   |  | "" |
| routerActiveOptions    | { exact: boolean }   |  | {exact: false} |
| routerTarget    | string   |  <code>_blank</code><code>_self</code><code>_parent</code><code>_top</code><code>framename</code>  | <code>_self</code>  |
| routerClass    | string   |  |  |

------
