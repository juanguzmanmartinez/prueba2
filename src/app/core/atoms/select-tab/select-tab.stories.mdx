### Props
| Property | Type  | Values | Default  |
| ------ | --------- |------ | --------- |
| selectTabList    | string[]   |  | |
| selectTabValue    | string   |     ""    |  ""  |
| selectTabDisabled      | boolean  | true/false       | false     |

### Events
| Event | Arguments  | Description |
| ------ | --------- |------ |
| change    |     | Emitted when select's value changes |

# Usages

Import InputsModule into your custom module

```typescript
import { SelectTabModule } from './select-tab.module';

@NgModule({
    imports: [SelectTabModule]
})

export class YourModule {}
```
